# Original 25 Sudoku Puzzles (000001-000025)

## How These Were Recovered

Every puzzle on onemil.xyz is a transformed copy of one of 25 base puzzles.
Puzzle ID mod 25 tells you which base puzzle it comes from (mod 0 = #25).
Each copy has two independent transforms applied:

**1. Position transform** (rows, columns, bands, stacks)
- Base-6 odometer with 8 levels controlling row/col/band/stack permutations
- Plus a 90° rotation every k steps (k differs per puzzle, e.g. k=23 for #1)
- Step number n = ceil(ID / 25)
- Fully solved by Kaleokai + Plurmorant + daddidecember
- Verified 6,990/6,990 puzzles — 100% correct

**2. Digit permutation** (relabeling 1-9)
- Each puzzle has its own permutation function with period N! (720 or 5040)
- At n ≡ 1 mod period, the digit permutation is the IDENTITY (no relabeling)
- We found IDs where this holds using 75,000 community-collected PDFs
- Key IDs: 252001-252025 all have n=10081, and 10081 mod 5040 = 1, 10081 mod 720 = 1
- So reversing just the position transform gives the true original

**Verification:**
- 6,990 puzzles: every position-reversed grid is a consistent digit relabeling of our original
- 48 independent identity-point PDFs across 6 different ID ranges all produce the exact same grid
- 8 puzzles with period 720 have 3-5 independent confirmations each
- 100% confirmed, both positions AND digits
